
'replicate_code.R': This file contains all the code required to reproduce the figures presented in the main manuscript.

'data_files': This folder includes all datasets used to generate the figures in the main manuscript.

'Conservation_effects_estimated_from_RCT.rds': This file is the estimated conservation coefficients, which is used for projecting electricity savings in the study.

'Projected_ac_ownership.xlsx': This file provides the projected air conditioner (AC) ownership data in China by 2030 under various SSP and RCP scenarios.

'Projected_cooling_electricity.xlsx': This file contains the projected cooling electricity consumption in China by 2030, based on different SSP and RCP scenarios.
